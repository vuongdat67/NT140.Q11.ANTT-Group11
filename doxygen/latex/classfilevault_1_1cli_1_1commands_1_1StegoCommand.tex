\doxysubsection{Detailed Description}
Steganography command for embedding/extracting data in images. 

Supports LSB (Least Significant Bit) steganography for hiding data within image files without visible changes.

Features\+:
\begin{DoxyItemize}
\item Embed secret data into PNG/\+BMP images
\item Extract hidden data from stego images
\item Calculate embedding capacity
\item Configurable bits per channel (1-\/4) 
\end{DoxyItemize}
