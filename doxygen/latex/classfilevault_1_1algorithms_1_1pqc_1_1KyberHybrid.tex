\doxysubsection{Detailed Description}
Hybrid encryption combining \doxylink{classfilevault_1_1algorithms_1_1pqc_1_1Kyber}{Kyber} with AES-\/\+GCM. 

This provides defense-\/in-\/depth by combining\+:
\begin{DoxyItemize}
\item \doxylink{classfilevault_1_1algorithms_1_1pqc_1_1Kyber}{Kyber} for post-\/quantum secure key encapsulation
\item AES-\/256-\/\+GCM for symmetric encryption
\end{DoxyItemize}

Even if one algorithm is broken, the other provides protection.
