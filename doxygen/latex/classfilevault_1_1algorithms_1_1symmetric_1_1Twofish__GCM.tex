\doxysubsection{Detailed Description}
Twofish-\/256-\/\+GCM implementation. 

Twofish is a 128-\/bit block cipher designed by Bruce Schneier, John Kelsey, Doug Whiting, David Wagner, Chris Hall, and Niels Ferguson. It was an AES finalist and is known for its conservative design.

References\+:
\begin{DoxyItemize}
\item Twofish\+: A 128-\/Bit Block Cipher \href{https://www.schneier.com/academic/twofish/}{\texttt{https\+://www.\+schneier.\+com/academic/twofish/}}
\item NIST SP 800-\/38D\+: GCM Mode \href{https://nvlpubs.nist.gov/nistpubs/Legacy/SP/nistspecialpublication800-38d.pdf}{\texttt{https\+://nvlpubs.\+nist.\+gov/nistpubs/\+Legacy/\+SP/nistspecialpublication800-\/38d.\+pdf}}
\end{DoxyItemize}

Security\+:
\begin{DoxyItemize}
\item Key size\+: 256 bits (32 bytes)
\item Block size\+: 128 bits (16 bytes)
\item Nonce size\+: 96 bits (12 bytes) for GCM
\item Tag size\+: 128 bits (16 bytes) for authentication
\end{DoxyItemize}

Features\+:
\begin{DoxyItemize}
\item Pre-\/computed S-\/boxes for performance
\item Key-\/dependent S-\/boxes for security
\item Supports 128, 192, and 256-\/bit keys
\item Patent-\/free and royalty-\/free
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This implementation uses Botan\textquotesingle{}s Twofish/\+GCM mode 

Twofish-\/256 provides equivalent security to AES-\/256 
\end{DoxyNote}
