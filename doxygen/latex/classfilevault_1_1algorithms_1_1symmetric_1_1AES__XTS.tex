\doxysubsection{Detailed Description}
AES-\/\+XTS encryption algorithm. 

Features\+:
\begin{DoxyItemize}
\item Designed for disk/sector encryption
\item Uses two keys (effectively double key size)
\item Tweak value provides sector-\/level randomization
\item No ciphertext expansion (same size as plaintext)
\item Not suitable for streaming data 
\end{DoxyItemize}
