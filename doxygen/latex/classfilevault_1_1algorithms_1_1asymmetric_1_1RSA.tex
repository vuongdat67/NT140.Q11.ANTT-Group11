\doxysubsection{Detailed Description}
\doxylink{classfilevault_1_1algorithms_1_1asymmetric_1_1RSA}{RSA} asymmetric encryption algorithm. 

Features\+:
\begin{DoxyItemize}
\item Public key encryption, private key decryption
\item Key sizes\+: 2048, 3072, 4096 bits
\item OAEP padding (PKCS\#1 v2.\+1) for encryption
\item PSS padding for signatures
\end{DoxyItemize}

Note\+: \doxylink{classfilevault_1_1algorithms_1_1asymmetric_1_1RSA}{RSA} encrypts limited data (key\+\_\+size -\/ padding overhead). For large data, use hybrid encryption (\doxylink{classfilevault_1_1algorithms_1_1asymmetric_1_1RSA}{RSA} + symmetric cipher).
