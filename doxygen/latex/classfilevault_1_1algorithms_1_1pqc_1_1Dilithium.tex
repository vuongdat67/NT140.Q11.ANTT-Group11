\doxysubsection{Detailed Description}
\doxylink{classfilevault_1_1algorithms_1_1pqc_1_1Dilithium}{Dilithium} Digital Signature Algorithm. 

NIST FIPS 204 (ML-\/\+DSA) -\/ Post-\/quantum secure digital signatures

\doxylink{classfilevault_1_1algorithms_1_1pqc_1_1Dilithium}{Dilithium} is a lattice-\/based signature scheme that provides\+:
\begin{DoxyItemize}
\item EUF-\/\+CMA security
\item Resistance to quantum computer attacks
\item Efficient signing and verification
\end{DoxyItemize}

Variants\+:
\begin{DoxyItemize}
\item Dilithium2\+: \texorpdfstring{$\sim$}{\string~}128-\/bit post-\/quantum security
\item Dilithium3\+: \texorpdfstring{$\sim$}{\string~}192-\/bit post-\/quantum security (recommended)
\item Dilithium5\+: \texorpdfstring{$\sim$}{\string~}256-\/bit post-\/quantum security 
\end{DoxyItemize}
