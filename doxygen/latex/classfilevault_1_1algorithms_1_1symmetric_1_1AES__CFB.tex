\doxysubsection{Detailed Description}
AES-\/\+CFB encryption algorithm. 

Features\+:
\begin{DoxyItemize}
\item Self-\/synchronizing stream cipher
\item No padding required
\item Can recover from bit errors (self-\/healing)
\item Encryption and decryption use same operation 
\end{DoxyItemize}
