\doxysubsection{Detailed Description}
Hybrid encryption using \doxylink{classfilevault_1_1algorithms_1_1asymmetric_1_1ECDH}{ECDH} + AES-\/\+GCM. 

Combines the security of ECC key exchange with the efficiency of symmetric encryption for encrypting arbitrary-\/sized data.

Encryption flow\+:
\begin{DoxyEnumerate}
\item Generate ephemeral \doxylink{classfilevault_1_1algorithms_1_1asymmetric_1_1ECDH}{ECDH} key pair
\item Derive shared secret with recipient\textquotesingle{}s public key
\item Use KDF to derive AES key from shared secret
\item Encrypt data with AES-\/\+GCM
\item Output\+: ephemeral public key + nonce + ciphertext + tag 
\end{DoxyEnumerate}
