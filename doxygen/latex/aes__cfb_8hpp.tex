\doxysubsection{Detailed Description}
AES-\/\+CFB (Cipher Feedback) mode encryption. 

CFB mode is a stream cipher mode that turns a block cipher into a self-\/synchronizing stream cipher. It\textquotesingle{}s similar to CBC but encrypts the previous ciphertext block instead of XORing.
