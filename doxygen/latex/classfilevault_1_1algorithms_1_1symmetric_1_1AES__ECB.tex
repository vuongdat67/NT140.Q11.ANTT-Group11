\doxysubsection{Detailed Description}
AES-\/\+ECB encryption algorithm. 

SECURITY WARNING\+: ECB mode is insecure!
\begin{DoxyItemize}
\item Identical plaintext blocks produce identical ciphertext blocks
\item Patterns in plaintext are visible in ciphertext
\item Use CBC, CTR, or GCM instead for real security
\end{DoxyItemize}

Only use for\+:
\begin{DoxyItemize}
\item Educational purposes
\item Single block encryption
\item Legacy system compatibility 
\end{DoxyItemize}
