\doxysubsection{Detailed Description}
LSB (Least Significant Bit) Steganography. 

Embeds secret data into image pixels by modifying the least significant bits. Supports PNG and BMP images.

Capacity\+: For RGB images, \texorpdfstring{$\sim$}{\string~}3 bits per pixel (if using 1 bit per channel) For an 800x600 image\+: (800 \texorpdfstring{$\ast$}{*} 600 \texorpdfstring{$\ast$}{*} 3) / 8 = 180,000 bytes max
