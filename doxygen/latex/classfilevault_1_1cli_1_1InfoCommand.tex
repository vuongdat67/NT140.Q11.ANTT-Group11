\doxysubsection{Detailed Description}
Info command -\/ display encrypted file metadata. 

Shows information about encrypted files without decrypting\+:
\begin{DoxyItemize}
\item File size (original, encrypted, compression ratio)
\item Encryption algorithm
\item KDF type and parameters
\item Compression type
\item Timestamp (if available) 
\end{DoxyItemize}
