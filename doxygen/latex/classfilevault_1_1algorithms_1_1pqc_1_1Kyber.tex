\doxysubsection{Detailed Description}
\doxylink{classfilevault_1_1algorithms_1_1pqc_1_1Kyber}{Kyber} Key Encapsulation Mechanism (KEM). 

NIST FIPS 203 (ML-\/\+KEM) -\/ Post-\/quantum secure key encapsulation

\doxylink{classfilevault_1_1algorithms_1_1pqc_1_1Kyber}{Kyber} is a lattice-\/based KEM that provides\+:
\begin{DoxyItemize}
\item IND-\/\+CCA2 security
\item Resistance to quantum computer attacks
\item Efficient key generation, encapsulation, and decapsulation
\end{DoxyItemize}

Variants\+:
\begin{DoxyItemize}
\item Kyber-\/512\+: \texorpdfstring{$\sim$}{\string~}128-\/bit post-\/quantum security
\item Kyber-\/768\+: \texorpdfstring{$\sim$}{\string~}192-\/bit post-\/quantum security (recommended)
\item Kyber-\/1024\+: \texorpdfstring{$\sim$}{\string~}256-\/bit post-\/quantum security 
\end{DoxyItemize}
