\doxysubsection{Detailed Description}
AES-\/\+OFB encryption algorithm. 

Features\+:
\begin{DoxyItemize}
\item True stream cipher mode
\item No error propagation (bit flip affects only one bit)
\item Keystream can be pre-\/computed
\item No padding required 
\end{DoxyItemize}
