\doxysubsection{Detailed Description}
Cha\+Cha20-\/\+Poly1305 AEAD encryption. 

Cha\+Cha20-\/\+Poly1305 is a modern AEAD cipher combining\+:
\begin{DoxyItemize}
\item Cha\+Cha20 stream cipher (designed by Daniel J. Bernstein)
\item Poly1305 MAC for authentication
\end{DoxyItemize}

Benefits over AES-\/\+GCM\+:
\begin{DoxyItemize}
\item No need for AES-\/\+NI hardware acceleration
\item Constant-\/time implementation easier to achieve
\item Better performance on software-\/only platforms
\item Recommended by IETF (RFC 8439)
\end{DoxyItemize}

Security\+:
\begin{DoxyItemize}
\item 256-\/bit key
\item 96-\/bit nonce (12 bytes) -\/ MUST be unique per encryption
\item 128-\/bit authentication tag (16 bytes)
\item Provides confidentiality and authenticity 
\end{DoxyItemize}
