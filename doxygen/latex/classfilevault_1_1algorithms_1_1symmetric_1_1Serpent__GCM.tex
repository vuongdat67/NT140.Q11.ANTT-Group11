\doxysubsection{Detailed Description}
Serpent-\/256-\/\+GCM implementation. 

Serpent is a 128-\/bit block cipher designed by Ross Anderson, Eli Biham, and Lars Knudsen. It was an AES finalist and is known for its high security margin.

References\+:
\begin{DoxyItemize}
\item Serpent\+: A Candidate Block Cipher for the AES \href{https://www.cl.cam.ac.uk/~rja14/serpent.html}{\texttt{https\+://www.\+cl.\+cam.\+ac.\+uk/\texorpdfstring{$\sim$}{\string~}rja14/serpent.\+html}}
\item NIST SP 800-\/38D\+: GCM Mode \href{https://nvlpubs.nist.gov/nistpubs/Legacy/SP/nistspecialpublication800-38d.pdf}{\texttt{https\+://nvlpubs.\+nist.\+gov/nistpubs/\+Legacy/\+SP/nistspecialpublication800-\/38d.\+pdf}}
\end{DoxyItemize}

Security\+:
\begin{DoxyItemize}
\item Key size\+: 256 bits (32 bytes)
\item Block size\+: 128 bits (16 bytes)
\item Nonce size\+: 96 bits (12 bytes) for GCM
\item Tag size\+: 128 bits (16 bytes) for authentication
\end{DoxyItemize}

\begin{DoxyNote}{Note}
This implementation uses Botan\textquotesingle{}s Serpent/\+GCM mode 

Serpent-\/256 provides very high security margin 

Slower than AES on most platforms but more secure 
\end{DoxyNote}
